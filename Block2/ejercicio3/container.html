UO251436 - Ejercicio 3 "Nuevos estándares en la web".

El propósito del ejercicio es mandar un mensaje a un servidor de imágenes "falso".

Una vez recibido el mensaje, el servidor de imágenes responderá con la url de una imagen que se mostrará a continuación (las imágenes se obtienen de picsum).

El mensaje enviado será el timestamp actual (sólo los segundos), que el servidor utilizará para devolver una url u otra.
